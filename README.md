# HTML-de-apresenta-o

Apresentacao web do sistema ControlX, pronta para publicacao pelo GitHub Pages.

## Como publicar no GitHub Pages

1. Acesse o repositorio no GitHub.
2. Entre em **Settings > Pages**.
3. Em **Build and deployment**, escolha **Deploy from a branch**.
4. Selecione a branch `main` e a pasta `/ (root)`.
5. Salve. O GitHub vai gerar a URL publica em alguns instantes.

## Arquivos principais

- `index.html`: entrada padrao para hospedagem.
- `apresentacao-sistema-completo-atualizada.html`: apresentacao completa.
- `bem-amigos-terminou.mp3`: audio do slide final.
- `foto-nos-3.jpeg`: foto usada no slide final.
